# Procurement Request Management Tool

This is a simple desktop application built with Python and Tkinter for managing purchase requests.

## Features

- Create purchase requests
- Save data into Excel
- Calculate total price automatically
- Display saved requests in a table
- Mark requests as urgent
- Track request status

## Technologies

- Python
- Tkinter
- OpenPyXL
- Excel

## Use Case

This tool can be used by purchasing or supply chain departments to record and track internal purchase requests.
